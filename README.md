# 🚀 TypeRush – Test de Vitesse de Frappe

## 📖 Description

TypeRush est une application web développée en **HTML5**, **CSS3** et **JavaScript Vanilla** permettant de mesurer la vitesse de frappe d'un utilisateur.

L'application affiche un texte aléatoire selon un niveau de difficulté choisi. L'utilisateur doit recopier le texte le plus rapidement et le plus précisément possible. À la fin du test, l'application calcule automatiquement le temps écoulé, la vitesse de frappe (WPM), la précision et affiche un message de performance.

Le meilleur score est enregistré localement grâce à **LocalStorage**.

---

# ✨ Fonctionnalités

* Sélection de trois niveaux de difficulté

  * Facile
  * Moyen
  * Difficile

* Bibliothèque de textes aléatoires (5 textes minimum par niveau)

* Démarrage automatique du chronomètre

* Coloration en temps réel

  * Vert : caractère correct
  * Rouge : caractère incorrect

* Calcul automatique :

  * Temps
  * WPM (Words Per Minute)
  * Taux de précision

* Affichage des résultats

* Message de performance

* Bouton **Nouveau Test**

* Sauvegarde du meilleur score avec LocalStorage

* Interface Responsive

# 📊 Calcul du WPM

Le calcul de la vitesse suit la formule officielle :

```
WPM = (Nombre de caractères corrects / 5)
      ÷
      Temps en minutes
```

---

# 🎯 Calcul de la précision

```
Précision =
(Caractères corrects
/
Caractères saisis)
×100
```
